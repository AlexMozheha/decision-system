Система прийняття рішень для оцінки фінансових ризиків підприємства.

Уніфікація Сирих Значень

Усі об'єктивні сирі значення (наприклад, гроші, час) повинні бути приведені до однієї найменшої одиниці для уникнення помилок округлення. Наприклад, якщо ціна вводиться у гривнях та копійках, вказуйте все у копійках (100 грн = 10000 копійок).

Об'єктивні vs. Суб'єктивні Фактори

"OBJECTIVE" (Об'єктивний): Вимірюється фізичними величинами. Ви вводите "Сире Значення" (Raw Value). Поле "Score" буде заблоковано.
"SUBJECTIVE" (Суб'єктивний): Оцінюється експертом. Ви вводите "Score" (Бал). Поле "Raw Value" буде заблоковано.

Напрямок Фактора (Зростає?)

Зростає? = true: Більше – краще (наприклад, Якість, Продуктивність).
Зростає? = false: Менше – краще (наприклад, Ціна, Час доставки).
